Overview

Flashcard Trainer is a simple browser-based study tool. It reads flashcards from a CSV file and lets you review them one by one.

Each card has a flashcard question and answer. Cards can also include an optional topic and optional multiple-choice content for MC practice.

CSV file format

Your CSV file must include a header row and at least one data row. Question and Answer are required; Topic is optional. Multi-choice mode is available when the header includes MC-Question, MC-Answer, and at least one MC-Distractor-* column. Additional distractor columns are optional.

Supported header layout

Use this header layout for best compatibility:

Topic,Question,Answer,MC-Question,MC-Answer,MC-Distractor-1,MC-Distractor-2,MC-Distractor-3

Required columns

Question
The flashcard question shown on the front of the card.
Answer
The flashcard answer shown on the back of the card.

Optional columns

Topic
An optional topic or grouping label shown above the question.
MC-Question
The multiple-choice question shown on the front when Multi-choice mode is active. To enable Multi-choice mode, the CSV header must include this column, MC-Answer, and at least one MC-Distractor-* column.
MC-Answer
The correct answer used in Multi-choice mode. This column is required only when the deck is intended to support Multi-choice mode.
MC-Distractor-1, MC-Distractor-2, MC-Distractor-3
Incorrect answer options used in Multi-choice mode. At least one distractor column must be present in the header to enable Multi-choice mode. Additional distractor columns are optional.

Example

"Topic","Question","Answer","MC-Question","MC-Answer","MC-Distractor-1","MC-Distractor-2","MC-Distractor-3"
"Web","What is HTML?","A markup language for web pages","Which technology defines the structure of a web page?","HTML","CSS","JavaScript","SQL"
"Web","What is CSS?","A stylesheet language","Which language controls the visual style of a web page?","CSS","HTML","JavaScript","Python"

Column names are not case-sensitive. Empty rows are ignored. Rows without both Question and Answer are skipped. A deck can contain both standard flashcards and cards with Multi-choice content.

Uploading a CSV file

  1. Select Upload CSV.
  2. Choose a CSV file from your device.
  3. The first card is shown after the file is successfully loaded.
  4. If the file cannot be read, check that it includes the required Question and Answer columns.

Reviewing flashcards

Click or tap the card to flip between the question and answer. Use the previous and next buttons to move through the deck.

On the answer side, use the status button to mark the card as correct, incorrect, or unanswered.

  • ✓ Correct
  • ✗ Incorrect
  • ○ Unanswered

The status line shows your current position, the number of correct and incorrect answers, and your current score.

Navigation filters

The Hide options let you skip cards based on their current answer status.

  • Correct hides cards marked as correct.
  • Incorrect hides cards marked as incorrect.
  • No Mark hides unanswered cards.

At least one status group must remain visible, so the app prevents all three groups from being hidden at the same time.

Multi-choice mode

Flashcard and Multi-choice are separate study modes. Flashcard mode studies all imported rows. Multi-choice mode becomes available when the CSV header includes MC-Question, MC-Answer, and at least one MC-Distractor-* column. The distractor columns do not all need to be present; one is sufficient. When Multi-choice mode is enabled, only MC-capable rows are shown.

The front of each MC card shows MC-Question followed by the correct MC answer mixed with distractors in random order. The back shows the correct option only. Distractors should be plausible but clearly incorrect for the specific question.

Generating CSV files with AI

Flashcard Trainer includes an integrated AI-powered workflow to create CSV files directly from training material. You can access this from the empty card state or the User Guide.

Quick start: From the empty card

When no deck is loaded, the empty card shows two quick links:

  • Generate one with the AI prompt — Opens the Create a Deck with AI dialog to build a new deck from training material.
  • View CSV format requirements — Opens this guide at the CSV file format section.

To create a deck with AI, follow these steps:

  1. Click Generate one with the AI prompt to open the Create a Deck with AI dialog.
  2. Follow the three-step workflow to create and save a new CSV file.

Three-step workflow

Step 1: Open the AI Prompt

Click Open AI Prompt to view the prompt template that guides your AI assistant.

  • Copy the prompt text directly into your AI chat.

Step 2: Generate CSV output

  1. Paste the AI prompt into your AI assistant.
  2. Replace the [INSERT CONTENT HERE AS TEXT, URLs OR BOTH] section with training text, one or more URLs, or both.
  3. Keep the required flashcard fields in every row: Question and Answer. Add MC fields only for rows that should support Multi-choice mode.
  4. Submit the prompt to the AI assistant.
  5. Copy the CSV output from the AI assistant.

Step 3: Create and save a CSV file

  1. Return to the dialog if needed.
  2. Paste the AI-generated CSV output into the AI-generated CSV output field.
  3. Enter a deck filename (defaults to my-ai-deck.csv).
  4. Click Save CSV File.
  5. The file downloads to your device.
  6. Upload the file using the Upload CSV button to start studying.

Source material

You can provide:

  • Training notes or documentation copied as text.
  • One or more URLs containing the training material.
  • A combination of pasted text and URLs.

For URLs, use links to the actual training material pages. A general homepage or landing page is unlikely to produce useful results because the prompt does not ask the AI to recursively crawl an entire website.

Notes & limitations

CSV content

  • The app expects a CSV file with a header row.
  • The Question and Answer columns are required.
  • Topic is optional.
  • Multiple-choice support is optional, but requires MC-Question, MC-Answer, and at least one MC-Distractor-* column in the header.
  • Additional distractor columns are optional.
  • Cards with a missing Question or Answer are ignored.
  • A deck can mix flashcard-only rows with flashcard-plus-MC rows.
  • Very large files may take longer to process in the browser.

AI-generated CSV files

  • The quality of generated flashcards depends on the quality and completeness of the source material supplied to the AI.
  • Different AI assistants may produce different results from the same prompt.
  • Review AI-generated flashcards before using them for study or assessment.

URL limitations

  • Use URLs that point directly to the relevant training content.
  • Provide multiple URLs if the training material spans several pages.
  • Avoid homepages, landing pages, and general navigation pages.
  • The prompt does not instruct the AI to recursively explore or crawl an entire website.
  • Some AI assistants may be unable to access URLs that require authentication, subscriptions, VPN access, or organisational permissions.

FAQ

What CSV columns are required?

The required columns are Question and Answer. Topic is optional. Multiple-choice support is optional, but if used the header must contain MC-Question, MC-Answer, and at least one MC-Distractor-* column.

Do column names have to use exact capitalisation?

No. Column names are matched without case sensitivity, so Question, question, and QUESTION are treated the same.

How do I create a CSV file using AI?

From the empty card, click the "Generate one with the AI prompt" link. Alternatively, you can access the AI Deck Creator from the User Guide. Follow the three-step workflow: open the AI prompt, paste it into your AI assistant with your training material, then paste the generated CSV into the File Creator and save it.

What source material can I provide to the AI?

You can provide pasted training text, one or more URLs, or a combination of both.

Can I provide multiple URLs?

Yes. Use multiple URLs when the training material is spread across several specific pages.

Can I provide only a website homepage?

No. For best results, provide URLs for the actual training material pages. The prompt does not ask the AI to recursively crawl an entire website from a homepage.

Why did the AI generate poor or incomplete results?

The source material may be incomplete, the URLs may be inaccessible, or the URLs may point to general navigation pages rather than actual training content.

Should I review AI-generated flashcards?

Yes. Review AI-generated flashcards before using them, especially if the material is important or used for assessment.
