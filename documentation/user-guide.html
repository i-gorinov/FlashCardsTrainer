Quick Start

  1. Click Upload CSV.
  2. Select a CSV file containing your flashcards.
  3. The study session starts automatically after the file is loaded.
  4. Read the question shown on the card.
  5. Click or tap the card to reveal the answer.
  6. On the answer side, mark your result as Unanswered, Correct or Incorrect.
  7. Move through the deck using the Previous and Next buttons.
  8. Monitor your progress and score in the status bar.

Tip: If you are using the app for revision, keep Shuffle Cards enabled so you do not learn the order of the cards.

CSV File Format

The CSV file must contain a header row and at least one flashcard row. The required columns are Question and Answer.

Column names are matched by header name, not by position, so columns may appear in any order.

Optional Columns

Category

The optional Category column displays a category label above the question. This is useful for larger decks that cover multiple topics.

Multiple-Choice Distractors

Columns beginning with Distractor are used to generate multiple-choice options. Distractors are incorrect answers shown alongside the correct answer.

Question,Answer,Distractor1,Distractor2,Distractor3
What is HTML?,Markup language,Database,Protocol,Operating System

Any number of distractor columns may be supplied. The Multiple Choice option appears only when at least one card contains distractors.

Studying Flashcards

Question Side

The front of the card displays the question text. If the CSV includes a category, the category appears above the question. If Multiple Choice mode is enabled, the answer options are also shown on the question side.

Answer Side

Click or tap the card to flip it and reveal the answer. In Multiple Choice mode, the answer side shows the correct option letter and the answer text.

Navigation

Use the navigation buttons to move through the deck.

ButtonAction
❮Go to the previous available card.
❯Go to the next available card.

Navigation respects any active filters. For example, if Hide Correct is enabled, cards marked Correct are skipped.

Study Options

Shuffle Cards

When Shuffle Cards is enabled, cards are presented in random order and are not repeated during the session. Changing this setting restarts the current session.

Multiple Choice

The Multiple Choice option appears only when the loaded CSV contains distractor columns. When enabled, the app displays answer options on the question side and reveals the correct option when the card is flipped.

Marking Answers

After reviewing the answer, click the circular status indicator on the answer side of the card. Each click cycles through the available answer states.

  • ○ Unanswered: no mark has been applied.
  • ✓ Correct: you answered correctly.
  • ✗ Incorrect: you answered incorrectly.

Note: Answer status can only be changed while viewing the answer side of the card.

Progress Tracking

The status bar shows your current card position, the number of cards marked Correct, the number of cards marked Incorrect and your overall score.

Desktop Format

Question 12/100 | Correct: 8 | Incorrect: 2 | Score: 8%

Mobile Format

12/100 | ✓8 | ✗2 | ★8%

Score Calculation

The score is calculated as:

Correct Answers ÷ Total Cards × 100

Unanswered cards are included in the total card count, so they reduce the score until they are marked Correct.

Filtering Cards

Filters let you focus on specific groups of cards. Filtered cards remain in the deck but are skipped during navigation.

  • Hide Correct: skips cards marked Correct.
  • Hide Incorrect: skips cards marked Incorrect.
  • Hide No Mark: skips cards that are still Unanswered.

Common Review Scenarios

Review Incorrect Answers Only

Enable Hide Correct and Hide No Mark.

Review Unanswered Cards Only

Enable Hide Correct and Hide Incorrect.

Review Correct Answers Only

Enable Hide Incorrect and Hide No Mark.

Note: The application limits filter combinations to prevent all cards from being hidden at the same time.

Resetting a Session

Click Reset to restart the current study session using the loaded CSV file and current study options.

Resetting a session will:

  • Clear all answer markings.
  • Reset progress statistics.
  • Rebuild the study order.
  • Return to the first card in the new session.

The loaded CSV file remains available and does not need to be uploaded again.

Error Messages

You may see an error message if the selected CSV file cannot be used. Common causes include:

  • The CSV file does not contain a header row and at least one data row.
  • The required Question or Answer column is missing.
  • The CSV contains malformed quoted text.
  • No valid question and answer pairs can be extracted.

Best Practices

  • Use Shuffle Cards for revision and recall practice.
  • Mark every answer honestly to keep the score meaningful.
  • Use filters to focus on weaker areas after the first pass through the deck.
  • Add categories to organise larger decks by topic.
  • Add distractors to create exam-style multiple-choice practice.
  • Reset and repeat sessions until you consistently reach your target score.

Notes and Limitations

  • Progress is not permanently saved.
  • Reloading the page starts a new session.
  • Answer markings are retained only for the current browser session.
  • Multiple Choice mode is available only when distractor columns exist.
  • The application runs in the browser and does not require server-side processing.

Frequently Asked Questions

Does column order matter?

No. The application identifies columns by their header names, so columns may appear in any order.

Are column names case-sensitive?

No. Header names are matched without regard to letter case, so Question, question and QUESTION are treated the same.

Why does the Multiple Choice option not appear?

The loaded CSV file does not contain any usable distractor values. Add columns such as Distractor1, Distractor2 and Distractor3 if you want to use Multiple Choice mode.

Can I save my progress?

No. Progress, score and answer markings are available only during the current browser session.

Why is the Next or Previous button disabled?

There may be no available card in that direction, or the remaining cards may be hidden by the active filters.

What happens when I click Reset?

The study session restarts. All answer markings and progress statistics are cleared, but the loaded CSV file remains available.

Is my CSV file uploaded anywhere?

No. The CSV file is read and processed locally in your browser.
